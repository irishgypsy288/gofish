================================================================================
                        WINDOW SPECIFICATIONS
================================================================================

Default Size:    1024 x 768 pixels
Minimum Size:    800 x 600 pixels
Resizable:       Yes
Frame Rate:      60 FPS
Card Size:       80 x 120 pixels
Card Spacing:    10 pixels
Button Size:     200 x 50 pixels

================================================================================
                        KEYBOARD SHORTCUTS
================================================================================

Welcome Screen:
- Enter/Space:  Start new game
- Escape:       Exit application

Gameplay Screen:
- Escape:       Exit application
- (Game plays automatically)

End Game Screen:
- Enter/Space:  Start new game
- Escape:       Exit application

================================================================================
                        MOUSE INTERACTIONS
================================================================================

Buttons:
- Hover:        Button highlights
- Click:        Button activates
- Cursor:       Changes to pointer over buttons

Cards:
- Display only: No interaction (AI vs AI mode)
- Future:       Could add click to select in human mode

================================================================================
                        STATE TRANSITIONS
================================================================================

┌─────────────┐
│   WELCOME   │
└──────┬──────┘
       │ Click "NEW GAME" or press Enter
       ↓
┌─────────────┐
│  GAMEPLAY   │
└──────┬──────┘
       │ Game ends (no more valid moves)
       ↓
┌─────────────┐
│  END GAME   │
└──────┬──────┘
       │ Click "NEW GAME" or press Enter
       ↓
┌─────────────┐
│  GAMEPLAY   │  ← Loop continues
└─────────────┘

Exit from any state: Click "EXIT" or press Escape
